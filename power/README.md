# ACS Screenshot Kiro Power

This directory contains the Kiro Power definition for the ACS Screenshot MCP server.

## What is a Kiro Power?

A Kiro Power packages MCP servers, documentation, and workflows into a reusable capability that can be installed in Kiro IDE.

## Files

- `POWER.md` - Main documentation and metadata
- `mcp.json` - MCP server configuration
- `steering/` - Workflow guides (optional)

## Installation

Users can install this power in Kiro IDE by:

1. Browsing the Kiro Powers registry
2. Searching for "ACS Screenshot"
3. Clicking "Install"

Or by adding the MCP server directly to their Kiro configuration.
